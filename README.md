# A simple spline app

## Installation

You will need a recent version of [Node.js](https://nodejs.org/en) and either npm or [bun](https://bun.com/)

// Set up and debug

`bun install`

`bun run dev`

// Code styling

`bun run lint`

`bun run format`

// Deployment

`bun run build`

`deploy.sh`
